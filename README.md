## Requirements 
```
python-3
pip
Internet Connection
And some other python packages
``` 
### Install Requirements (on Linux/Termux)

```python
>> pip insall pytube
```
### INSTALLATION >
```python
apt update && apt upgrade -y
pkg install git
pkg install python
pip install requests
rm -rf YT-Downloder
git clone --depth=1 https://github.com/U7P4L-IN/YT-Downloder.git
cd YT-Downloder
chmod +x *
python YT.py
```
